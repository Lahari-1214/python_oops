#  Python OOPS Concepts

This repository contains **Object-Oriented Programming (OOPS) concepts in Python**
with **simple explanations and examples**.

It is designed for:
- Beginners
- Interview preparation
- Revision and practice

---

##  Concepts Covered

- Class & Object
- Constructor (`__init__`)
- `self` keyword
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Special (Dunder) Methods
- Real-world OOPS examples

---

##  Folder Structure

Each folder represents one OOPS concept with clear examples.
